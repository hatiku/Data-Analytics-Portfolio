# Customer Churn Analysis and Prediction

## Project Overview

This project analyzes customer churn behavior and builds a machine learning model to predict customers who are likely to leave the company. The project combines Python, SQL Server, Power BI, and Machine Learning techniques to provide actionable business insights.

---

## Business Problem

Customer churn significantly affects company revenue and profitability. The company wants to:

- Understand why customers leave.
- Identify high-risk customers.
- Build an interactive dashboard.
- Predict future churn.

---

## Project Objectives

- Analyze customer churn patterns.
- Build an interactive dashboard.
